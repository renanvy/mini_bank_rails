== Mini bank

Pequeno exemplo de um mini banco implementado com Rails.

Demo: <tt>https://mini-bank-rails.herokuapp.com/</tt>

Para rodar o projeto:

* <tt>bundle install</tt>

* <tt>rake db:create</tt>

* <tt>rake db:migrate</tt>

* <tt>rails server</tt>

Para rodar os testes:


* <tt>rspec spec/</tt>
